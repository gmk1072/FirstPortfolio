Meteors Mini-Game

This mini-game was designed in Processing, a sketchbook like version of Java with many more implementations for visual styling and creation. It was all designed from the ground up in processing using some borrowed assets, like the music and a couple images. I programmed everything for this game. It makes use of various random algorithms, bounding box collision detection, and force vectors for movement.
This project was meant to be a simple asteroids look-alike. It was a nice insight into creating a certain feel of gameplay using forces, sound effects and different mechanics.
